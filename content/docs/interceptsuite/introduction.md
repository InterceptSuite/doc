# Welcome to InterceptSuite Documentation

**InterceptSuite v1.2.0** -Advanced SOCKS5 proxy for TLS interception with Python extension support. Perfect for security professionals, developers, system administrators, and network engineers who need powerful network analysis capabilities.

## About InterceptSuite

InterceptSuite is a high-performance SOCKS5 proxy that sits between your applications and the network. It intercepts, decrypts, and allows you to inspect or modify traffic in real-time -including TLS-encrypted connections -without modifying the applications themselves.

### Key Features

- **TLS Interception** -Advanced TLS traffic analysis and manipulation
- **DTLS Interception** -Datagram TLS traffic analysis and manipulation *(Professional)*
- **TCP/UDP Interception** -Raw TCP and UDP traffic interception and analysis
- **Universal TLS Upgrade Detection** -Automatic detection of protocol upgrades to TLS *(Professional)*
- **Python Extension Support** -Extensible with custom Python 3 scripts
- **SOCKS5 Proxy** -High-performance proxy server with full SOCKS5 protocol support
- **Cross-Platform** -Windows, macOS, and Linux support
- **Project Management** -Organize your work with project files *(Professional)*
- **PCAP Export** -Export captured traffic for external analysis in Wireshark *(Professional)*

## Platform Support

### Cross-Platform

InterceptSuite runs natively on Windows, macOS (Apple Silicon), and Linux. A single download page provides installers for all supported platforms.

### Open Source

The Standard Edition is completely free and open source, with full core feature support.

### Professional

The Professional Edition adds advanced features designed for serious security research and professional network analysis workflows.
