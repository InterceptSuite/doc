# License Activation

Learn how to activate your InterceptSuite Professional Edition license.

## Overview

InterceptSuite ships in two editions:

- **Standard Edition (Free)** -Includes all core features with community support
- **Professional Edition** -Includes all Standard features plus advanced professional capabilities

The Professional Edition requires a valid license key to unlock advanced features.

## Getting a License Key

1. Visit [interceptsuite.com](https://interceptsuite.com/)
2. Select your preferred plan and complete the payment process. Make sure to enter your correct email address.
3. You will receive an email with payment confirmation
4. Navigate to the [customer portal](https://polar.sh/interceptsuite/portal/)
5. Log in with the same email used during payment
6. Retrieve your license key from the customer portal

## Downloading the Professional Edition

If you haven't yet downloaded the Professional Edition:

1. Visit [interceptsuite.com/download](https://interceptsuite.com/download)
2. Download the Professional Edition installer for your platform
3. Follow the [Installation Guide](/docs/installation) for platform-specific instructions

## Activating Your License

### Auto Activation (Recommended)

When you open InterceptSuite Professional Edition for the first time:

1. A license key window will appear requesting your key
2. Enter your license key in the input field
3. Click **Submit** -an internet connection is required
4. If the key is valid, you will proceed to the project file selection window

### Manual Activation (Offline)

If you don't have internet access or prefer offline activation:

1. In the license key window, select **Manual Activation**
2. Enter your license key in the input field
3. Click **Submit** -you will receive offline activation data
4. Copy the offline activation data
5. Go to [interceptsuite.com/activate-license](https://interceptsuite.com/activate-license)
6. Paste the activation data and submit
7. Copy the response from the website
8. Paste the response back into InterceptSuite and click **Submit**
9. You now have access to all Professional features

### Important Notes

- Internet connection required for online activation (manual activation available for offline use)
- This is a one-time process -you only need to activate once per installation
- License validation happens in real-time

## Verifying Activation

To confirm your license is active:

1. **Successful first launch** -If activation succeeded, you proceed to the project file selection window
2. **Application title** -The app should display "InterceptSuite Pro" in the title bar
3. **Professional features** -You have access to all Professional Edition features

## Professional Edition Features

Once activated, you gain access to:

- **Project File Management** -Organize your work with project files
- **PCAP File Export** -Export captured traffic for external analysis in Wireshark
- **Universal TLS Upgrade Detection** -Automatic detection of protocol upgrades to TLS

## Troubleshooting

### License Key Not Working

1. **Check the key format** -Ensure you copied the entire key, including any dashes
2. **Verify internet connection** -Online activation requires active internet access
3. **Check your email** -Make sure you're using the correct key from your purchase email
4. **Contact support** -If the issue persists, email [support@interceptsuite.com](mailto:support@interceptsuite.com) with your order details
